Why paste CSS? — CSS gives exact hex values (e.g. #DC0018) → WCAG contrast ratios calculated to 2 decimal places. Without CSS, colours are estimated from screenshot pixels — less precise, especially for similar shades.
In Figma Dev Mode: select a frame → Copy → Copy as CSS. In browser: DevTools → Elements → Styles → copy relevant rules.
